# Web-Scraper-Cognifyz
A powerful Python web scraper developed for Cognifyz Technologies using BeautifulSoup and Requests. It extracts structured data from websites, processes HTML efficiently, handles robots.txt, retries, and exports results to CSV and JSON. Designed for ethical, efficient, and real-world data scraping.
